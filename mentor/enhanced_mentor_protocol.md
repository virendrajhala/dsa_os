# Enhanced Mentor Protocol Compatibility File

The authoritative mentor protocol is `mentor/mentor_protocol.md`.

This file exists only for backward compatibility with older prompts and tooling
that refer to `mentor/enhanced_mentor_protocol.md`. It is not a separate
protocol fork and holds no protocol content of its own. Everything it used to
duplicate — the Implementation Blueprint, the post-code review order, the
revision requirements, and the scoring split — now lives solely in
`mentor/mentor_protocol.md`. If anything ever refers here for those rules, read
`mentor/mentor_protocol.md` instead.
